# 一些CLI终端操作
- 检查Python版本
```
python --version
```
- 进入Python解释器
```
python
```
- 进入某写好的脚本的交互界面
```
python -i example.py
```
## 管理文件
- 列出文件夹下的所有文件和文件夹
```
ls
```
文件有长度length，文件夹没有。可以以此区分
-  更换目录/所在文件夹
```
cd <目标路径>
```
- 转移到想去的文件夹位置：绝对路径或当前目录下的相对路径
```
cd ~
```
- 返回home directory
```
cd Desktop
```
桌面文件夹刚好是home directory下的Desktop
- 返回上一级目录/文件夹
```
cd ..
```
- 创建文件夹
```
mkdir <文件夹名字>
```
- 移动文件(夹)
```
mv <文件/文件夹> <目标路径>
```

# 内置数值运算函数
- `abs()`
- `divmod(x,y)`：返回元组，商和余数同时返回，比如`>>> divmod(5,2)`返回`(2,1)`
- `pow(a,b)`
- `round(a,b)`：对a保留b位小数，并四舍五入；没有`b`就直接四舍五入整数值
- 数字类型的转换，可以用`type(x)`返回数字`x`的类型
	- `int()`
	- `float()`
	- `complex()`：增加虚部
