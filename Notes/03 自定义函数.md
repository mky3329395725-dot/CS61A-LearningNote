可以说函数就是一个很强大的抽象事物的方法
# 定义函数的方法
> 定义函数：名称 →绑定→ 复合操作

```
def <function name> (<formal parameters>):
	return <statement>
```
`def ... :`**函数签名**，定义名称与所使用的参数（形参）
	名称与形参，构成了其形成的新环境的框架。传入实参则是为这个框架填入实体
`return ...`**函数体**，函数真正操作的表达式。return语句必须缩进。函数体已经与该函数名称绑定。
# 特性
- 内置函数（built-in）、模块导入函数、自定义函数的使用方式都一样，单独使用起来没有标识表明其来源
- 函数定义时，函数体还没有被执行，直到被调用
	- Squirreled，被保存起来，藏在环境里
# 调用自定义函数
- 调用时其在计算机中运行的流程
	1. 生成一个局部框、新环境
	2. 将传入的参数与函数的形参 bind 起来
	3. 在新环境中执行函数体，并且返回return值
# 纯函数与非纯函数
用Print与None，引入Pure Function和Non-Pure Function
> Python会自动**返回**输入的任何表达式的值
> None这个内容不会显示任何值/内容

## 纯函数Pure Funtion
只有函数的**映射**功能。也就是给定输入，经过函数得到输出，不产生副作用
- 不依赖外部变量
- 无副作用（包括修改外部状态）
- 相同输入相同输出
## 非纯函数
- 可能有**副作用**（包括修改外部状态）
- 或以来外部变量
```
print(print(3))
# print函数本身的返回值/输出是None
# 但print函数还有一个副作用是显示输入的内容

# 因此，在这个代码里，外层print调用后，为了求解第一层括号内的内容
# 第一层括号内部的print(3)被调用：
# 1. 显示第二层括号内的3
# 2. 返回None给第一层属于外面这个print的括号

# 首先明确：
# None发送给Python解释器，Python视其为<空>。None = 什么都没有，Python解释器不会返回任何值/内容
# 显示 ≠ 返回值，显示就是不管这是个什么东西，直接展示出来

# 3. 由于print会显示输入的任何值，直接显示内部的返回内容None
# 虽然None这四个字母代表的值是空的，但是它的内容是None，因此被展示

# 外面这层print显示了内层print返回的None，本身返回内容也是None
# 4. 这次的None不会再有显示
# 因为这次None不是要显示了，而是作为空值被发给Py解释器，不会有任何反应
```
